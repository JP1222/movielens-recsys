# Data Directory

Place the downloaded MovieLens datasets here. Recommended layout:

- `data/raw/ml-1m/ratings.dat` and `movies.dat`
- `data/raw/ml-32m/ratings.csv` and `movies.csv`
- `data/artifacts/` (generated by the offline pipeline)

The offline scripts will create the `artifacts` directory automatically when you run:

```bash
python -m scripts.cli --ratings data/raw/ml-1m/ratings.dat --movies data/raw/ml-1m/movies.dat --output-dir data/artifacts
```
